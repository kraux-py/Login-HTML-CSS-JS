# Login Animate

Login Animate is a modern login page template with animated transitions built using HTML, CSS, and JavaScript.

## About

Login Animate provides a sleek and user-friendly interface for user authentication. It features smooth transitions between sign-in and sign-up forms, along with social media login options.

## Features

- Smooth animations: The page incorporates smooth transitions between sign-in and sign-up forms using CSS animations.
- Social login integration: Users can log in using their social media accounts with provided icons for Google, Facebook, GitHub, and LinkedIn.
- Responsive design: The login page is designed to be responsive, ensuring a seamless experience across various devices and screen sizes.

## Installation

1. Clone the repository: `git clone https://github.com/your_username/login-animate.git`
2. Navigate to the project directory: `cd login-animate`

## Usage

1. Open the `index.html` file in your web browser.
2. You can toggle between the sign-in and sign-up forms by clicking the "Sign In" and "Sign Up" buttons.
3. Enter your credentials or register for an account, and click the respective buttons to proceed.
